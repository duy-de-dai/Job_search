## Cách Sử Dụng

- Chạy file `index.py` để tạo index mới nếu có data mới

    ```bash
    $ python index.py
    ```

- Chạy `search.py` để thực hiện truy vấn tìm kiếm

    ```bash
    $ python search.py "Tìm kiếm của bạn"
    ```
